# Landing Page Copy Guidelines

## Voice & Tone
- Direct, confident, empathetic
- Second person ("you/your") throughout
- Active voice, short sentences
- No jargon, no corporate-speak
- Feel: "a smart friend who's been through this before"

## Hero Section
- H1: Short, punchy, action-oriented (e.g., "Land Your Next Job")
- Subheading: Use `<br>` tags for deliberate 3-line visual rhythm
- Focus on transformation, not features
- NO pricing in the hero — keep it focused on the value proposition
- Trust pills show key stats (session count, duration, availability)

## Problem Section
- Lead with a stat that validates the reader's frustration
- Use sup tags for source citations
- 3 pain points, each with icon + headline + 1-2 sentence description
- Pain points should feel personally relatable, not abstract

## Solution Section
- Eyebrow: "What You'll Build" — focus on outcomes, not features
- Headline ties count to promise: "6 sessions. Everything you need to land the job."
- Each deliverable is concrete and tangible (not "learn about X" but "build X")

## Pricing Copy Patterns
- Price anchoring: always show 3 tiers vertically (total value → regular → discounted)
- Strikethrough on higher prices, green highlight on the active price
- Discount code appears in 9+ locations throughout the page:
  1. Nav area (implicit via CTA)
  2. Value stack section (code box)
  3. Pricing card (discount box)
  4. Pricing card CTA button
  5. Final CTA paragraph text
  6. Final CTA button text
  7. Mobile sticky CTA
  8. FAQ answers (when relevant)
  9. Comparison table header (price in column name)
- CTA buttons with price: "Enroll Now, <s>$699</s> $599" or "Enroll Now, <s>$699</s> $599 with CODE"
- Keep code name prominent: bold, green color, in `<code>` tags

## FAQ Writing Rules
- Always widen the audience, never exclude
  - Bad: "This is only for tech professionals"
  - Good: "The bootcamp is designed for anyone searching for a professional role: operations, marketing, sales, consulting, finance, tech, and more."
- Clarify permanent vs temporary access explicitly
  - "You keep permanent access to all session recordings... After the 3 months, you can continue your Leland+ subscription..."
- Address both the "can't make it" and "what if I don't like it" objections early
- Technical skill FAQ: affirm both non-technical and technical audiences

## Testimonial Selection
- Choose reviews that mention specific outcomes or frameworks
- Include the coach name they're reviewing
- Bold the headline quote, italic the full review
- Keep reviews from coaches who are actually on the page when possible (flag for update if coaches change)

## Coach Card Copy
- Tagline = their session topic (e.g., "Session 1: Build Your AI Job-Search Engine")
- Tags = credentials (company + education), not job titles
- Base profile link, no category suffix on URLs

## Comparison Table Copy
- Use specific details, not just checkmarks (e.g., "6 sessions, 90 min each" not just "Yes")
- Include real competitor names and real prices
- Highlight what's unique to Leland (AI tools, platform membership, community)

## Calls to Action
- Primary: "Enroll Now" + price
- Exploratory: "See Curriculum" (links to #curriculum anchor)
- Always include reassurance below final CTA: "All sessions recorded · Full refund after Session 1 if not satisfied"

## Numbers & Formatting
- Use specific numbers over vague claims ("59% of job seekers" not "most job seekers")
- Cite sources with superscript numbers, list sources at page bottom
- Monetary values: $599 format (no decimals unless cents)
- Time: "90 min" in pills, "90 minutes" in body text

## Discount Code Naming Convention
- Format: MONTHNAME + AMOUNT (e.g., MARCH100)
- Updates monthly or per cohort — must be changed in all 9+ locations
